Ciclo De Vida De un Proyecto

Planificación:

Es donde se definirá las metodologías para llevar acabo un proyecto determinado, por lo general con etapas definidas, y con recursos designados.

Análisis:

Recopila y formula los requisitos del cliente y examina cualquier restricción que se pueda aplicar. Se definen las Metas y objetivos y se indaga las necesidades y restricciones del cliente

Diseño:

En esta parte se empieza con el desarrollo de los requisitos del cliente, tanto lógicos como gráficos

Implementación:

Una vez claro el diseño la funcionalidad del sistema, se empieza a desarrollar la lógica del código a implementar

Pruebas:

las pruebas son de gran importancia ya que con esto podemos ver los posibles errores que nuestro software tenga y así poder evaluarlo que tan bueno es y si se adapta a las necesidades del cliente

Documentación

En la documentación se plasman paso a paso las funcionalidades de nuestro software y las recomendaciones para una optima utilización

Despliegue:

Esta fase se implementa cuando nuestro software ha pasado todas las pruebas requeridas y se evalúa en un ambiente de produccion

Mantenimiento:

El mantenimiento consta de brindar un soporte y actualizaciones adecuadas para que nuestro software no quede obsoleto
